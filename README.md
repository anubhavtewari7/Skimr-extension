# Skimr: The Sage Academic Assistant 🌿⚡

**Skimr** is a premium, keyless Chrome extension designed to transform how students and researchers interact with digital content. It converts dense academic articles, PDFs, and YouTube lectures into interactive study modules instantly—**100% free and without any API limits.**

## 🚀 The Vision
Most AI tools require expensive monthly subscriptions or complex API key setups. **Skimr** breaks that barrier. By utilizing public AI endpoints and localized DOM extraction, Skimr provides a "Tony Stark" style academic HUD that is accessible to every student, everywhere, for free.

## ✨ Core Features

### 🎥 Silent YouTube Intelligence
Summarize hour-long lectures in seconds. Skimr silently intercepts hidden video transcripts directly from the browser memory, providing comprehensive notes without needing to open the transcript panel.

### 🎮 Active-Recall Quiz Mode
Turn your notes into a game. The interactive Quiz Mode uses spaced-repetition logic to test your knowledge with a distraction-free, "Swipe-to-Score" interface.

### 🖱️ Contextual "Explain This"
Stuck on a complex sentence? Highlight any text on any website, right-click, and select **"Skimr: Explain this"** for an instant, simplified breakdown in your sidepanel.

### 📚 Auto-Citation Engine
Skimr automatically generates APA/MLA-style citations for every page or video you scan, ensuring your bibliographies are always ready to go.

### 🖨️ OneNote-Style Export
Export your saved "Vault" notes into beautiful, printable PDFs designed with a premium Microsoft OneNote aesthetic—complete with blue ruled lines and clean typography.

## 🛠️ Technology Stack
- **AI Core:** Powered by the Pollinations AI text endpoint (Keyless & Unlimited).
- **Frontend:** Vanilla JavaScript, HTML5, and "Liquid Glass" CSS design.
- **Manifest:** Manifest V3 (Latest Chrome Extension Standard).
- **Privacy:** 100% local processing. No user data is sent to external servers except for the text being analyzed by the AI.

## 📦 Installation (Developer Mode)
1. Clone this repository: `git clone https://github.com/anubhavtewari7/Skimr-extension.git`
2. Open Chrome and navigate to `chrome://extensions/`
3. Enable **Developer Mode** (toggle in the top-right).
4. Click **Load unpacked** and select the `extension` folder from this repo.
5. Click the Skimr icon in your toolbar to ignite the scanner!
